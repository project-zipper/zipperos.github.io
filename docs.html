ZipperOS Documentation

Installation

1. Download the ISO from the Download page.
2. Flash it using BalenaEtcher, Rufus, or dd.
3. Boot from USB and follow the installer instructions.

System Requirements

• 2GB RAM (4GB recommended)
• 20GB storage
• 64-bit CPU
• Optional: touchscreen or stylus support
